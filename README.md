# cloned
 A Bit Parody [fighting] game

It will be awesome :)

##To play:
  1. Launch game
  2. Click on Arcade
  3. Select your character

##Controls
  * Forward: Right Arrow
  * Back:    Left Arrow
  * Punch:   Space
  * Kick:    Control
  * Jump:    Up Arrow



Featuring Music by Matthew Pablo (Blackmoor Tides)
