# cloned
 A Bit Parody [fighting] game
 
 - Sarah Baucom
 - Monis Khan
 - Benny Xiong
 - Jorge Ramos

It will be awesome :)

##To play:
  1. Launch game
  2. Select game mode using arrow keys and `Enter` (or game pad controller 1)
  3. Select your character (press 'Kick' button to lock in selection)

##Keyboard Controls:
###Player 1
  * Right: Right Arrow
  * Left:    Left Arrow
  * Punch:   Control
  * Kick:    Space
  * Jump:    Up Arrow
  * Block:   Z
  * Special: X
  * Pause: Esc
  * Cheats:
    * Invincibility, 10x damage, and unlimited special: G

###Player 2
  * Right: D
  * Left:  A
  * Punch: P
  * Kick:  O
  * Jump:  W
  * Block: I
  * Special: U
  * Pause: Esc

##Game Pad Controls:
###Player 1 and 2
  * Right:   Left analog stick right
  * Left:    Left analog stick left
  * Punch:   X
  * Kick:    A
  * Jump:    B
  * Block:   Y
  * Special: Right Trigger
  * Pause: Start
